# Text Stream Clustering
Кластеризация потоковых текстовых данных. Удобно для кластеризации новостей.

Для качественной кластеризации необходимо подобрать хорошую sentence-transformers-модель и, по возможности, дообучить её на своих данных. Различные методы обучения описаны [здесь](https://sbert.net/).

Пример использования: `example.ipynb`.
